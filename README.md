# 🚀 Sonar Rock vs Mine Prediction using Machine Learning

## 📖 Project Overview

This project uses **Logistic Regression** to classify underwater sonar signals as either **Rock (R)** or **Mine (M)**. The model is trained on the Sonar dataset containing 60 numerical features extracted from sonar signals.

---

## 🎯 Objective

Develop a machine learning model that accurately predicts whether an underwater object is a rock or a mine based on sonar signal patterns.

---

## 📂 Dataset Information

* **Dataset:** Sonar Dataset
* **Total Records:** 208
* **Features:** 60 Numerical Features
* **Target Classes:**

  * **R** → Rock
  * **M** → Mine
