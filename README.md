# Sonar-Rock-vs-Mine-Prediction-using-Machine-Learning
Machine Learning project to classify underwater sonar signals as Rock (R) or Mine (M) using Logistic Regression. Includes data preprocessing, exploratory analysis, model training, evaluation, and prediction.


# 🚀 Sonar Rock vs Mine Prediction

## 📌 Project Overview
This project predicts whether an underwater object is a **Rock** or a **Mine** using Machine Learning. The model is trained on the Sonar Dataset containing 60 numerical sonar signal features.

---

## 🎯 Objective
Build a binary classification model that accurately identifies underwater objects based on sonar signal frequencies.

---

## 📂 Dataset
- Sonar Dataset
- 208 Records
- 60 Numerical Features
- Target:
  - R = Rock
  - M = Mine

---

## 🛠️ Technologies Used

- NumPy

## 📊 Data Preprocessing
- Loaded dataset using Pandas
- Checked dataset shape

---

## 🤖 Machine Learning Algorithm
- Logistic Regression

---

## 📈 Model Training
- Train-Test Split (90% Training, 10% Testing)
- Stratified Sampling
- Random State = 42

---

## 📏 Evaluation Metrics
- Training Accuracy
- Testing Accuracy
- Accuracy Score

---

## 🔍 Prediction
The model predicts whether the sonar signal belongs to:
- Rock (R)
- Mine (M)
